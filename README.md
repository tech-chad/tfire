# Terminal Fire

## tfire
Fire simulation in the terminal using python and curses

### Commands
- **<kbd>Q</kbd>** **<kbd>q</kbd>** To quit.
- **<kbd>c</kbd>** Cycle through colors
- **<kbd>0 - 9</kbd>** Change the speed 0-Fast, 5-Default, 9-Slow
